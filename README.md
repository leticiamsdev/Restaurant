# L'ESSENCE - Restaurant Landing Page

A responsive landing page built for a fictional high-end restaurant, featuring a clean and elegant design. 

**Live Demo:** [https://leticiamsdev.github.io/Restaurant/](https://leticiamsdev.github.io/Restaurant/)

## Project Overview

This project is a static website created to practice building interfaces with HTML, CSS, and basic JavaScript. It simulates the online presence of a fine-dining establishment, including a hero section, a story section, a tasting menu showcase, and a simple reservation form layout.

### Tech Stack

- HTML5
- CSS3 (Vanilla)
- JavaScript (for mobile menu toggle)
- FontAwesome (for icons)

## Folder Structure

- `index.html`: The main HTML file.
- `styles/`: Contains the CSS files organized by section (e.g., `home.css`, `menu.css`, `reserve.css`) for better maintainability.
- `javascript/`: Contains `script.js` which handles the responsive navigation bar.
- `images/`: Static image assets used across the page.

## How to Run Locally

Since this is a simple static site, you don't need any build tools or local servers to run it.

1. Clone the repository:
   ```bash
   git clone https://github.com/leticiamsdev/Restaurant.git
   ```
2. Navigate to the project folder and double-click `index.html` to open it directly in your browser.
